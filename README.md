# FINAL MODIFICATION PROPOSAL

Our project aims to also involve web users to be able to input their own personal tusok-tusoks or tidbits of information. How will we do this? This first forum page will be made into effect in our website by use of CRUD or Create, Read, Update and Delete. By using the lessons and topics we learned in CS3 web development, we will make an effective, efficient and user-friendly forum page.

This aligns with our mission of providing info for the people, by the people.

And also, we always take into account user-friendliness by making a very aesthetic and organized layout that displays information effectively so that everyone can access even in mobile phones. This aligns with our second mission, accessible information is effective information.
